Methodology Improvements vs Old Backtest

  • Daily prices, not monthly — capture true volatility, not averaged-out smoothness
  • Fees deducted (0.25%) — Bitkub spot fee, applied to every buy
  • Real Bitkub BTC/THB — direct, no FX approximation
  • Monotonic BTC verified — unit test ensures we never "lose" BTC magically
  • Rolling distribution — show range of outcomes, not single cherry-picked path

Strategy Definitions

  • flat: ฿1,000/วันคงที่
  • sma2x: ฿1,000 ปกติ; ฿2,000 ตอนราคา < SMA(140 วัน, ≈20W)
  • mayer: Mayer Multiple = price/SMA(200). <0.8 → 3x, <1.0 → 2x, >2.4 → 0.5x
  • bollinger3: BB(140, 2σ). ใต้ lower → 3x, ในกรอบ → 1.5x, เหนือ upper → 1x
  • lumpsum: ลงทั้งก้อนวันแรก เท่ากับงบ flat DCA ทั้งช่วง

Known Data Quirks (Disclosed)

  • Bitkub API ส่ง 2 bars สำหรับ 2024-01-31, 2024-02-01, 2025-04-25
  • Fetcher dedupe โดย keep bar ที่ volume สูงกว่า (=complete daily aggregation)
  • Decision logged ใน data/btc_thb_daily.json::deduplicated
